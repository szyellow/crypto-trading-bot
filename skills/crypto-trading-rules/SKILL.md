# Crypto Trading Rules Skill

## 交易规则清单

### 1. 黑名单规则

#### 稳定币 (永久禁止)
- USDC
- USDT
- USDG
- DAI
- TUSD
- BUSD
- **规则**: 永久禁止买入，不需要释放机制

#### 止损币种 (条件释放)
- 触发止损后自动加入
- 需要连续3次趋势评分≥8分才能解除
- 例如: NEAR

### 2. 买入条件检查清单

#### 必须同时满足:
1. ✅ 不在黑名单
2. ✅ 趋势评分 ≥ 7分
3. ✅ 非横盘状态 (趋势3-5分且波动率<0.5%)
4. ✅ 持仓占比 < 阈值
5. ✅ 冷却期已结束
6. ✅ 波动率 ≥ 0.5%

### 3. 卖出条件

#### 止盈:
- 动态调整: 趋势8-10分 → 10-15%
- 趋势下降时下调止盈

#### 止损:
- 趋势10分: -3%
- 趋势8-9分: -2%
- 趋势6-7分: -1.5%
- 趋势≤5分: -1%

### 4. 仓位管理

- 单笔最大: $40
- 最大持仓: 5个
- 现金储备: 保持30%以上

### 5. 每日检查清单

- [ ] 检查黑名单是否需要更新
- [ ] 检查止损线是否正常
- [ ] 检查持仓是否超仓
- [ ] 检查止盈单是否正确挂单

---

## 重要提醒

**稳定币处理**:
- 发现稳定币被买入 → 立即卖出释放资金
- 稳定币占用资金无法增值

**横盘期处理**:
- 趋势3-5分 + 波动率<0.5% = 横盘
- 横盘期禁止买入

**止损线修复**:
- 收紧: -2.5% → -2.0% (更容易触发)
- 放宽: -2.5% → -3.0% (更难触发)
