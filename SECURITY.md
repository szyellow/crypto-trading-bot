## 报告安全漏洞

如果您发现了安全漏洞，请不要通过公开 issue 报告。请通过以下方式私下联系我们：

- 发送邮件至：security@example.com
- 在邮件主题中注明 "[Security] Crypto Trading Bot"

我们会在 48 小时内回复，并尽快修复漏洞。

## 安全最佳实践

### API 密钥管理

- 永远不要将 API 密钥提交到代码仓库
- 使用环境变量存储敏感信息
- 定期轮换 API 密钥
- 限制 API 密钥权限（只读、交易等）

### 运行环境

- 在安全的网络环境中运行
- 使用防火墙限制访问
- 定期更新依赖包
- 监控异常交易活动

### 资金安全

- 只投入您能承受损失的资金
- 设置合理的止损线
- 定期检查账户余额和交易记录
- 启用交易所的双重认证
