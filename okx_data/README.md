# AI 自主加密货币交易系统 v4.1

一个基于 OKX 交易所 API 的智能加密货币交易机器人，采用多 Agent 架构，整合**币市麻雀战法**、多维度共振策略、情绪分析和动态风控系统。

## 🎯 币市麻雀战法 v4.1 (时区感知版)

> **核心理念**: "麻雀虽小，五脏俱全" - 追求小而稳的盈利，见好就收，积少成多。

### 策略特点
- **日目标**: $9 (1%盈利目标)
- **周目标**: $21 (2.5%盈利目标)
- **止盈**: +3% (麻雀见好就收)
- **止损**: -0.5% (严格止损)
- **持仓时间**: 15分钟-2小时 (快进快出)
- **单笔仓位**: $12-$15 (时区动态调整)
- **时区感知**: 6个交易时段动态调整策略参数

## 🏗️ 多 Agent 架构

本系统采用模块化多 Agent 设计，各组件职责明确，协同工作：

- **主交易程序**: `ai_trading_bot.js` - 核心决策引擎
- **币市麻雀战法**: `config-sparrow.js` - 时区感知策略配置
- **多维度共振**: `strategy-resonance.js` - 四维共振分析
- **策略进化**: `strategy-evolution.js` - 自动优化参数

## 🚀 主要特性

### 时区感知系统
6个交易时段动态调整策略参数：

| 时段 | 时间 | 活跃强度 | 仓位建议 |
|------|------|----------|----------|
| 亚盘早盘 | 00:00-04:00 | ⭐⭐ | $10-12 |
| 亚盘午盘 | 04:00-08:00 | ⭐⭐⭐⭐⭐ | $12-15 |
| 欧盘早盘 | 08:00-12:00 | ⭐⭐⭐⭐⭐ | $15-18 |
| 欧盘午盘 | 12:00-16:00 | ⭐⭐⭐⭐⭐ | $15-20 |
| 美盘早盘 | 16:00-20:00 | ⭐⭐⭐⭐⭐ | $15-18 |
| 美盘午盘 | 20:00-00:00 | ⭐⭐⭐⭐ | $12-15 |

### 动态止盈止损
根据趋势评分自动调整：

| 趋势评分 | 止损 | 止盈 |
|---------|------|------|
| ≤5分 | -1% | 2% |
| 6-7分 | -1.5% | 3% |
| 8-9分 | -2% | 10% |
| ≥10分 | -3% | 15% |

## 📊 策略配置

核心配置文件 `config-sparrow.js`：

```javascript
module.exports = {
  dailyTarget: 9,      // 日目标 $9
  weeklyTarget: 21,    // 周目标 $21
  takeProfit: 0.03,    // 止盈 3%
  stopLoss: -0.005,    // 止损 -0.5%
  timezoneAware: true, // 启用时区感知
};
```
