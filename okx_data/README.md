# AI 自主加密货币交易系统 v4.1

一个基于 OKX 交易所 API 的智能加密货币交易机器人，采用多 Agent 架构，整合**币市麻雀战法**、多维度共振策略、情绪分析和动态风控系统。

## 🎯 币市麻雀战法 v4.1 (时区感知版)

> **核心理念**: "麻雀虽小，五脏俱全" - 追求小而稳的盈利，见好就收，积少成多。

### 策略特点
- **日目标**: \$9 (1%盈利目标)
- **周目标**: \$21 (2.5%盈利目标)
- **止盈**: +3% (麻雀见好就收)
- **止损**: -0.5% (严格止损)
- **持仓时间**: 15分钟-2小时 (快进快出)
- **单笔仓位**: \$12-\$15 (时区动态调整)
- **时区感知**: 6个交易时段动态调整策略参数
