# 自动交易配置 - 2026-02-14

## 交易参数设置

| 参数 | 数值 |
|------|------|
| **检查频率** | 每15分钟 |
| **运行时间** | 24小时不间断 |
| **单笔金额** | 2 USDC |
| **最大同时持仓** | 5笔（10 USDC） |
| **累计止损线** | 7 USDC（20%本金） |
| **保留资金** | 25 USDC |
| **胜率要求** | >70% |

## 每次扫描流程（15分钟周期）

### 第1步：市场筛选（0-2分钟）
- 扫描Polymarket热门市场
- 筛选当天/次日结算的市场
- 24h交易量 > $100K
- 记录候选市场

### 第2步：数据收集（2-7分钟）
- CoinDesk新闻抓取
- 链上数据检查（如有工具）
- 社交媒体情绪分析
- 整合多源信息

### 第3步：胜率评估（7-10分钟）
- 综合分析市场情绪
- 计算胜率估计
- 只有>70%才进入下一步

### 第4步：执行决策（10-15分钟）
- 评估是否满足所有条件
- Bankr全自动执行买入
- 记录交易详情
- 发送汇报消息

## 风险控制规则

1. **止损机制**
   - 累计亏损达7 USDC立即停止当天交易
   - 次日09:00重新评估后恢复

2. **仓位管理**
   - 最多5笔同时持仓
   - 超过则等待持仓结算后再开新仓

3. **数据源要求**
   - 每笔交易必须基于≥2个独立数据源
   - 单一数据源不执行

4. **胜率门槛**
   - 严格>70%胜率要求
   - 模糊或不确定的市场跳过
