# Crypto Trading Rules - 加密货币交易规则

## 核心规则

### 1. 黑名单管理

#### 稳定币（永久禁止）
- USDC
- USDT
- USDG
- DAI
- TUSD
- BUSD

**规则**: 稳定币永久禁止买入，不需要释放机制。

#### 止损币种（可释放）
- 触发止损后自动加入
- 需要连续2次趋势≥8分才能释放
- 释放后重新评估

#### 手动禁止（人工添加）
- 根据市场情况人工添加
- 需要手动移除

### 2. 买入条件

必须同时满足：
1. ✅ 趋势评分 ≥ 7分
2. ✅ 非横盘状态（趋势3-5分且波动率<0.5%）
3. ✅ 不在黑名单
4. ✅ 现金充足
5. ✅ 持仓未超限

### 3. 止损规则

| 趋势评分 | 止损线 |
|----------|--------|
| ≤5分 | -1% |
| 6-7分 | -1.5% |
| ≥8分 | -2%至-3% |

### 4. 止盈规则

动态调整：
- 趋势满分(10分): 止盈15%
- 趋势8-9分: 止盈10%
- 趋势6-7分: 止盈6%
- 趋势≤5分: 止盈2-3%

### 5. 资金分配

- 单笔最大: $40
- 最大持仓: 5个币种
- 现金储备: 至少30%

## 检查清单

每次交易前检查：
- [ ] 币种不在稳定币黑名单
- [ ] 币种不在止损黑名单（或已满足释放条件）
- [ ] 趋势评分≥7分
- [ ] 非横盘状态
- [ ] 现金充足
- [ ] 持仓未超限
